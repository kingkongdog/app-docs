# Support

Thank you for using Mouse Wheel Scroll Volume. If you encounter any issues while using the app, please refer to the following information for help.

## FAQ
1. **How do I set the modifier key?**  
   Open the app, go to Settings → Modifier Key, and select the key you want to use.

2. **Volume adjustment does not work?**  
   Make sure the app has the required accessibility permissions and try restarting the app.

3. **App crashes or errors?**  
   Try restarting your computer or reinstalling the app. If the problem persists, please contact us.
