# Mouse Wheel Scroll Volume

## Features
Mouse Wheel Scroll Volume is a lightweight Mac tool that allows you to quickly adjust system volume using your mouse wheel.

- Hold a custom modifier key + scroll the mouse wheel to change volume  
- Customize modifier keys and volume indicator position  
- High-performance mode with fast response  
- Local operations only, no network transmission, secure and private

## Use Cases
- Quickly adjust volume while watching movies or listening to music  
- Control volume without opening the system volume panel during multitasking  
- Suitable for all Mac users, supports Intel and Apple Silicon chips
